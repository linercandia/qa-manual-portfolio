# QA Manual Testing Portfolio

This repository contains my QA Manual testing portfolio, focused on functional testing, test case design, bug reporting, and basic API testing.
The goal of this project is to demonstrate my understanding of software quality assurance fundamentals and my ability to document and execute manual tests in a structured way.

## Project Scope

- Functional testing of a sample web application
- Test plan creation
- Manual test case design and execution
- Bug reporting and documentation
- Basic API testing using Postman

## Tools Used

- Test case design: Excel / Google Sheets
- Bug tracking: Jira-style reports (simulated)
- API testing: Postman
- Documentation: Markdown
- Version control: Git & GitHub

## Repository Structure

test-plan/ -> Test plans and testing strategy
test-cases/ -> Manual test cases
bug-reports/ -> Bug reports and defect documentation
api-testing/ -> Postman collections and API tests
evidence/ -> Screenshots and test evidence

## Project Status

This project is currently under development and will be updated with new test cases and improvements.
